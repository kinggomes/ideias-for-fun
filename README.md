Este é meu segundo projeto independente (sem guia ou acompanhamento de algum curso ou tutor de minha faculdade) posto em público.

Este projeto se trata de um seleção para recomendação de filmes, músicas, e jogos. Para que o usuário tenha uma experiência coerente e intuitiva, as páginas de recomendações possuem a mesma estrutura e estilização, a única diferença sendo as cores, que serão explicadas mais abaixo, O projeto é dividido em 4 partes:

- Página principal: Uma página simples para introdução, onde o usuário irá ler um pequeno texto informando sobre os benefícios do lazer e do entretenimento, e segundo isso, verá os botões representantes de cada catálogo logo após o texto, onde ele poderá escolher entre o de filmes, de músicas, ou jogos. Esta página foi feita puramente com HTML/CSS, e seu título é estilizado com as três cores das três páginas abaixo.

- Página de filmes: Uma página que contém as recomendações de filmes. Cada card representa um filme, com suas próprias informações (nome do filme, direção, e etc..) além de um botão que enviará o usuário para o trailer do filme selecionado. Na parte superior da página há o menu de navegação, onde o usuário encontrará botões para a página principal, para os outros catálogos, e também uma barra de pesquisa, onde o usuário poderá pesquisar pelo nome do filme, ou o seu gênero. Não é necessário escrever o nome do filme completo, pois a lógica criada para a barra de pesquisa permite que o filme seja identificado apenas com parte da informação inserida (por exemplo, caso o usuário digite apenas "aranha" e clique em "Procurar", o filme do Homem-Aranha já aparecerá). 
Esta página foi feita com HTML/CSS e com interação por JS. Suas cores principal são o preto e o vermelho, para referenciar a Netflix, um dos maiores streamings de filmes.

- Página de músicas: Uma página que contém as recomendações de músicas. Cada card representa uma música diferente, com suas próprias informações (nome da música, gênero, e data de lançamento) além de um botão que enviará o usuário para a música selecionada, em sua versão no YouTube. Na parte superior da página há o menu de navegação, onde o usuário encontrará botões para a página principal, para os outros catálogos, e também uma barra de pesquisa, onde o usuário poderá pesquisar pelo nome da música, seu cantor, ou seu gênero. Assim como na página de filmes, não é necessário escrever o nome do cantor ou da música em si perfeitamente, pois a barra de pesquisa permite que seja identificado apenas com parte da informação inserida. 
Esta página foi feita com HTML/CSS e com interação por JS. Suas cores principal são o preto e o verde, para referenciar o Spotify, um dos maiores streamings de músicas.

- Página de jogos: Uma página que contém as recomendações de jogos. Cada card representa um jogo, com suas próprias informações (nome do jogo, gênero, plataforma, e etc...) além de um botão que enviará o usuário para o site mais acessível em que possa comprar o jogo. Na parte superior da página há o menu de navegação, onde o usuário encontrará botões para a página principal, para os outros catálogos, e também uma barra de pesquisa, onde o usuário poderá pesquisar pelo nome do jogo, seu gênero, ou sua plataforma. Assim como nas páginas anteriores, não é necessário pesquisar a informação perfeitamente, pois a barra de pesquisa permite que seja identificado apenas com parte da informação inserida. 
Esta página foi feita com HTML/CSS e com interação por JS. Suas cores principal são o preto e o azul, para referenciar a Steam, um dos maiores aplicativos para acesso e compra de jogos.



Este projeto é um remake de uma atividade que realizei durante um curso de web design, onde, junto com um grupo, criamos um site com a mesma idéia. Porém, grande parte do projeto original foi criado com o framework Bootstrap, onde usamos templates e códigos já prontos para "copia e cola". Dessa vez, eu decidi recriar esse projeto totalmente do zero, sem utilizar nenhum framework, e tornando-o ainda mais robusto, adicionando mais cards, mais informações, e também barras de pesquisa. E, apesar dos títulos das páginas, elas não são uma tentativa de afirmar quais são os "Melhores Filmes/Músicas/Jogos de Todos os Tempos" objetivamente, e sim apenas uma seleção feita por mim, através de meu gosto pessoal.
